About This Project

This comprehensive health and wellness web application was developed as part of my 4-month internship program in 2020. The project showcases my skills in full-stack web development and demonstrates my ability to create complex, user-friendly applications.

Technologies Used

Frontend

HTML5, CSS3, JavaScript, Chart.js for data visualization, Font Awesome for icons

Backend

Node.js with Express.js for API development and server-side logic

Database

MongoDB with Mongoose for data persistence and user profiles

Key Features Implemented

  • User authentication system with secure password hashing
  • Advanced BMI calculator with personalized recommendations
  • Comprehensive nutrition tracking with food database integration
  • Customizable workout planner with exercise library
  • Water intake tracker with visual indicators
  • Interactive progress charts and data visualization
  • Responsive design that works on all device sizes
  • User settings and preferences system

Learning Outcomes

Through developing this application during my internship, I gained valuable experience in:

  • Full-stack JavaScript development
  • RESTful API design and implementation
  • Database design and management
  • User experience (UX) principles
  • Responsive web design techniques
  • Data visualization with Chart.js
  • Project planning and time management
